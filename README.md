## 这是什么

《煤龙拳》是一个**致敬《街霸2》玩法**的 2D 格斗游戏，整个游戏就是**一个 HTML 文件**——没有引擎、没有框架、没有任何第三方依赖，原生 JavaScript + Canvas 从零写起，连 BGM 和音效都是 WebAudio 实时合成的。

它是龙虾舰队折腾实录的一部分，全程用 **Claude Fable 5** 辅助手搓出来的。两个角色、一套 SF2 规格的对战机制、骨骼绘制的像素小人，塞进 87KB 的一个文件里。

## 特性

- **16 招** + 命中普通技可搓招**取消连招**
- 完整的必杀体系：波动 / 升龙 / 旋风（空中可用）/ 气满超必杀
- 投技 + 中段指令技
- **双人对战**（F2 切换）+ 单人对 CPU
- 三局两胜，投币、选人流程
- **触屏支持**：手机上自动切到虚拟摇杆 + 按键
- 街机感：CRT 扫描线 + 暗角滤镜，320×180 低清像素渲染
- WebAudio 现场合成的 BGM、KO / 连段 / 回合音效
- **零依赖、单文件**，双击 `index.html` 即玩，也能直接丢到任何静态托管

## 操作

**P1**：`WASD` 移动 ｜ `U` 轻拳 `I` 重拳 `J` 轻脚 `K` 重脚 ｜ 蹲+键=下段，空中=跳击 ｜ 近身前+重拳=投技，远身前+重拳=中段指令技

**P2**：方向键移动 ｜ 小键盘 `4` 轻拳 `5` 重拳 `1` 轻脚 `2` 重脚

**通用**：`Enter` 投币 ｜ `F2` 切双人

**必杀指令**：

| 招式 | 指令 |
|---|---|
| 波动 | ↓→ + 拳 |
| 升龙 | →↓→ + 拳 |
| 旋风（空中可用） | ↓← + 脚 |
| 超必杀（需气满） | ↓→↓→ + 拳 |

## 技术

- 纯原生 **JavaScript + Canvas**（800×450 输出，内部 320×180 低清像素）
- 角色用**骨骼方式逐帧绘制**，不依赖贴图素材
- **WebAudio** 程序化合成全部音效与 BGM
- 帧数据 / 判定框 / 取消窗口手写实现
- **无任何第三方库或构建步骤**——源码就是成品

## 怎么跑

直接双击 `index.html` 用浏览器打开即可。或者部署到任意静态托管（Cloudflare Pages / GitHub Pages / itch.io），把这个目录作为根目录、`index.html` 作为入口就行，无需构建命令。
